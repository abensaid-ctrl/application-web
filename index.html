📚 BiblioTech — À propos

Application de gestion de bibliothèque avec backend en C et base de données SQLite.

# Compilation
gcc server.c -o server -lsqlite3 -lpthread

# Lancement
./server 8080

# Ouvrir index.html dans le navigateur

Ce frontend fonctionne en mode démo sans serveur. Connectez le backend C pour la persistance.
