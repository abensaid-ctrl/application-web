# 📚 BiblioTech — Gestion de Bibliothèque

Application web complète avec **backend en C**, **SQLite** et interface moderne.

## Structure

```
bibliotheque/
├── server.c      ← Serveur HTTP en C (API REST)
├── index.html    ← Frontend (HTML/CSS/JS)
├── Makefile      ← Compilation
└── README.md
```

## Lancement rapide

```bash
# 1. Installer SQLite (Ubuntu/Debian)
sudo apt-get install libsqlite3-dev

# 2. Compiler
make

# 3. Lancer le serveur (port 8080)
./server 8080

# 4. Ouvrir index.html dans le navigateur
```

## Fonctionnalités

### 📖 Livres
- Ajouter / Supprimer des livres
- Gérer les exemplaires disponibles
- Filtrer par catégorie
- Recherche en temps réel

### 👥 Membres
- Inscription de membres
- Suivi des emprunts par membre
- Désactivation de membres

### 🔄 Emprunts
- Créer un emprunt (membre + livre + date de retour)
- Retourner un livre (stock mis à jour automatiquement)
- Détection des retards
- Filtrer par statut

### 🏠 Dashboard
- Statistiques en temps réel
- Emprunts récents
- Livres disponibles

## API REST (Backend C)

| Méthode | Route | Description |
|---------|-------|-------------|
| GET | /api/stats | Statistiques globales |
| GET | /api/livres | Liste des livres |
| POST | /api/livres | Ajouter un livre |
| DELETE | /api/livres/:id | Supprimer un livre |
| GET | /api/membres | Liste des membres |
| POST | /api/membres | Ajouter un membre |
| DELETE | /api/membres/:id | Désactiver un membre |
| GET | /api/emprunts | Liste des emprunts |
| POST | /api/emprunts | Créer un emprunt |
| POST | /api/emprunts/:id | Retourner un livre |

## Mode démo

L'interface fonctionne **sans serveur** avec des données de démonstration. 
Lancez le backend C pour activer la persistance réelle avec SQLite.
